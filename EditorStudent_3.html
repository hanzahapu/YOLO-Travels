The Quiz starts with a main page listing all the rules within the quiz. This main page also consist with some important information about Sri Lanka. By clicking on start quiz button the user will be navigated to the Quiz page. In the Quiz page the left bar contains the question numbers and the question is displayed in the body of the html. While the user is attempting a particular question, the related question number is highlighted with green colour. Also its a must to answer each and every question, if not a pop-up modal will appear saying to choose an option. At the same time before changing to the next question the Correct and incorrect answers will be highlighted with red and green. Finally after the quiz ceases the time spent, score, corrrect answers will be displayed with a relavent color and a relevant remark. By clicking on continue button it will proceed to the quiz summary which displays whether the answer was correct or incorrect with the relevant Question number. The user can only navigate within the Quiz summary modals after ending the Quiz.
